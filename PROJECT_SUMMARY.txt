================================================================================
                         1. PROJECT OVERVIEW
================================================================================

Serenity is an AI-powered mental wellness chatbot designed to provide 
empathetic emotional support and gentle coping strategies. It uses OpenAI's 
GPT-4.1 model to generate thoughtful, supportive responses in a safe, 
non-judgmental environment.

PURPOSE:
- Offer a safe space for users to talk about their feelings
- Provide active listening and emotional validation
- Suggest general coping and relaxation techniques
- Encourage seeking professional help when needed
- Provide crisis resources for users in immediate danger

IMPORTANT LIMITATIONS:
- NOT a licensed therapist or medical professional
- Cannot diagnose, treat, or prescribe medications
- Not a substitute for professional mental health care
- Not an emergency crisis intervention service

POST /api/chat
  Description: Main chat endpoint for AI responses
  Request Body:
    {
      "messages": [
        { "role": "user", "content": "..." },
        { "role": "assistant", "content": "..." }
      ]
    }
  Response:
    {
      "reply": "AI-generated response text"
    }
  Error Response:
    {
      "error": "Error message"
    }
  Process:
    1. Validates messages array
    2. Prepends system prompt
    3. Calls OpenAI responses.create()
    4. Extracts output_text or output[0].content[0].text
    5. Returns reply to frontend

GET /api/health
  Description: Health check endpoint
  Response:
    {
      "status": "ok",
      "timestamp": "2026-02-12T13:37:42.000Z"
